# Project 1 — FreshMart Retail Sales Dashboard

## Overview
A complete Excel analytics workbook analyzing 9,994 retail 
orders across 4 regions, 3 categories, and 4 years (2014-2017).

## Tools Used
- Microsoft Excel 365
- Functions: SUMIF, COUNTIF, IF, YEAR, TEXT, COUNTA
- Features: PivotTables, Charts, KPI Dashboard

## Key Business Insights
- West region has highest sales ($725K) and best margin (14.94%)
- Furniture category losing money — only 2.49% margin
- Technology = best category (17.4% margin)  
- Business grew 51.4% from 2014 to 2017
